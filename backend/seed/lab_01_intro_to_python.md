# Lab 1: Introduction to Python

## Objectives
- Understand basic Python syntax
- Write simple functions and loops
- Work with lists and dictionaries

## Exercises

### Exercise 1.1: Variables and Types
Create variables of different types: integer, float, string, and boolean. Print their types using `type()`.

### Exercise 1.2: Functions
Write a function `calculate_average(numbers)` that takes a list of numbers and returns the average.

### Exercise 1.3: Loops
Write a `for` loop that prints the multiplication table from 1 to 10.

## Tips
- Use `python3` to run your scripts
- Test each function individually
- Remember to handle edge cases (e.g., empty lists)
