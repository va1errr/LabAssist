# Lab 2: Data Structures

## Objectives
- Master Python's built-in data structures
- Understand when to use lists vs tuples vs sets vs dictionaries
- Practice nested data structures

## Exercises

### Exercise 2.1: List Operations
Given a list of student grades, write functions to:
- Find the highest and lowest grades
- Calculate the median
- Remove duplicates

### Exercise 2.2: Dictionary Manipulation
Create a dictionary representing a student record with keys: name, id, grades, email. Write functions to:
- Add a new grade
- Calculate GPA
- Export to JSON format

### Exercise 2.3: Sets
Given two lists of students enrolled in Math and Physics, find:
- Students taking both courses (intersection)
- Students taking only one course (symmetric difference)

## Tips
- Use list comprehensions where possible
- `set()` is your friend for uniqueness
- Remember dictionaries are unordered (pre-3.7)
